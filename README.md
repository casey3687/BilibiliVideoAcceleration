# Bilibili Playback Rate Menu

Chrome 扩展：增强 B 站视频播放器原生倍速菜单，提供 `0.25x` 到 `4x` 的倍速选项，步长为 `0.25x`。

## 本地安装

1. 下载release中的zip文件或者dist文件夹下的bilibili-playback-rate-menu-0.1.0.zip文件
2. 解压到你想要的位置
3. 打开 Chrome。
4. 进入 `chrome://extensions`。
5. 开启右上角开发者模式。
6. 点击加载未打包的扩展程序。
7. 选择解压后的文件夹：

   ```text
   C:\bilibili-playback-rate-menu-0.1.0
   ```

8. 打开或刷新 B 站视频页面。
9. 点击播放器原生倍速按钮，选择需要的倍速。
